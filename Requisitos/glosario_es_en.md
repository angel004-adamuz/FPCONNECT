# Glosario Tecnico (ES/EN)

- Autenticacion - Authentication: Proceso para verificar identidad de usuario.
- Autorizacion - Authorization: Control de permisos por rol.
- Backend: Capa servidor con logica de negocio y acceso a datos.
- Frontend: Capa cliente para interfaz e interaccion.
- ORM (Object-Relational Mapping): Capa que mapea objetos a tablas relacionales.
- Migracion - Migration: Cambio versionado del esquema de base de datos.
- Endpoint: Ruta HTTP expuesta por una API.
- Payload: Datos enviados en una peticion o respuesta.
- Socket: Canal de comunicacion bidireccional en tiempo real.
- JWT (JSON Web Token): Token firmado para autenticacion stateless.
- Refresh Token: Token de larga duracion para renovar access token.
- Hash: Transformacion criptografica unidireccional.
- Rate Limiting: Restriccion de frecuencia de peticiones por cliente.
- CORS (Cross-Origin Resource Sharing): Politica de acceso entre origenes web.
- Deployment - Despliegue: Puesta en produccion de la aplicacion.
- CI/CD: Integracion y despliegue continuo automatizado.
- Observabilidad - Observability: Capacidad de monitorear estado y errores.
